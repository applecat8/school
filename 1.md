## 内力的计算

1. 判断结构的静定性
    
    1. 将结点视为自由体，杆件视为约束。
    2. 4个结点,每个结点2个自由度，$n=4\times2=10$
    3. 8根杆,每个结点1个自由度，$c=8$
    4. 多余约束数$f=c-n=0$

    结果:分析可知，该桁架为无多余约束数的集合不变结构

2. 分析外力

    1. 对结点3取矩:
    $$\sum M_3=0$$
    $$P_5\times2+P_2\times1-P_1\times0.5-P_2\times1.5=0$$
    $$P_5=2.5$$
    2. 同理得,对结点5取矩:
    $$P_{3x}=0$$
    $$P_{3y}=2.5$$

3. 采用节点法求内力

    1. 分析节点5:
    * y方向平衡
    $$\sum F_{5y} = 0$$
    $$F_{5-2}\cos30^o = -P_5$$
    $$F_{5-2} = -\frac{5\sqrt{3}}{3}$$
    * x方向平衡
    $$\sum F_{5x} = 0$$
    $$F_{5-2}\sin30^o = -F_{5-4}$$
    $$F_{5-4} = \frac{5\sqrt{3}}{6}$$

    2. 分析结点2:
    * y方向平衡
    $$\sum F_{2y} = 0$$
    $$F_{2-5}\cos30^o+F_{2-4}\cos30^o=-P_1$$
    $$F_{2-4} = -5\sqrt{3}$$
    * x方向平衡
    $$\sum F_{2x} = 0$$
    $$F_{2-5}\cos60^o=F_{2-4}\cos60^o+F_{2-1}$$
    $$F_{2-1} = \frac{5\sqrt{3}}{3}$$

    3. 由于该结构为左右对称结构,所以结点1的分析与结点2相同，结点3的分析与结点5相同
    $$F_{3-4} = \frac{5\sqrt{3}}{6}$$
    $$F_{3-1} = -\frac{5\sqrt{3}}{3}$$
    $$F_{1-4} = -5\sqrt{3}$$

    4. 自此每个杆的内力都已求解完毕

4. 内力计算结果总结

    |杆|内力|
    |-|-|
    |杆1-2|$\frac{5\sqrt{3}}{3}$|
    |杆1-3|$-\frac{5\sqrt{3}}{3}$|
    |杆1-4|$-5\sqrt{3}$|
    |杆2-4|$-5\sqrt{3}$|
    |杆2-5|$-\frac{5\sqrt{3}}{3}$|
    |杆5-4|$\frac{5\sqrt{3}}{6}$|
    |杆3-4|$\frac{5\sqrt{3}}{6}$|

## 结点位移计算

> 结点的位移计算采用单位力法
> $$\Delta_ip=\sum\frac{N_iN_P}{EA}l_i$$

1. 结点3位移计算

* 因为结点3为固定绞支座，所以有:

$$\Delta _{3x} = 0$$
$$\Delta _{3y} = 0$$

2. 结点5位移计算

* 结点5为滑动支座，所以有:
$$\Delta _{5y} = 0$$

* x方向位移计算:
在结点5处作用一个水平向右的单位力,计算在该单位力作用下的内力分布，如图:
$$\Delta_{5x}=\frac{1}{EA}\big[\frac{5\sqrt{3}}{6}\times1+\frac{5\sqrt{3}}{6}\times1]$$
$$\Delta _{5x} = 1.3746\times10^{-6}$$

3. 结点2位移计算

* x方向位移计算:在结点2处做一个向右的水平单位力,计算在该单位力作用下的内力分布，如图:

$$\Delta_{2x}=\frac{1}{EA}\big[(-\frac{5\sqrt{3}}{3})\times\frac{1}{2}+\frac{5\sqrt{3}}{6}\times\frac{3}{4}+\frac{5\sqrt{3}}{6}\times\frac{1}{4}+(-\frac{5\sqrt{3}}{3})\times(-\frac{1}{2})+\frac{5\sqrt{3}}{3}\times\frac{1}{2}+(-5\sqrt{3})\times\frac{1}{2}+(-5\sqrt{3})\times(-\frac{1}{2})\big]$$

$$\Delta _{2x} = 1.3746\times10^{-6}$$

y方向位移计算:在结点2处做一个水平向上单位力,计算在该单位力作用下的内力分布，如图:

$$\Delta_{2y}=\frac{1}{EA}\big[(-\frac{5\sqrt{3}}{3})\times\frac{\sqrt{3}}{6}+\frac{5\sqrt{3}}{6}\times(-\frac{\sqrt{3}}{12})+\frac{5\sqrt{3}}{6}\times(-\frac{\sqrt{3}}{4})+(-\frac{5\sqrt{3}}{3})\times\frac{\sqrt{3}}{2}+\frac{5\sqrt{3}}{3}\times\frac{\sqrt{3}}{6}+(-5\sqrt{3})\times(-\frac{\sqrt{3}}{6})+(-5\sqrt{3})\times\frac{\sqrt{3}}{6}\big]$$

$$\Delta _{2y} = 1.5873\times10^{-6}$$
